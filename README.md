# Docker Compose - Jupyter Data Sciencie

This repository use official docker image jupiter and add docker-compose for use volume in your notebooks.

![Jupiter](http://jupyter.org/assets/main-logo.svg)

```
.
├── data
├── docker-compose.yml
├── jupyter
│   └── Dockerfile
├── notebooks
└── tree
```

## Run:

```sh
$ docker-compose up
```

## Requirements:

* Docker
* Docker-compose
